# Bike-Buyer-Sales-and-Demographic-Analysis
This data analysis project explores customer behavior and conversion rates using Excel . I designed a dynamic dashboard with real-time filtering to visualize how socio-economic factors influence purchasing decisions.

##Project Overview
This project involves a comprehensive analysis of customer demographic data bike sales. The goal was to identify key purchasing trends and behavioral patterns to help the marketing team target high-conversion customer segments. I cleaned the raw data, performed exploratory data analysis (EDA), and developed an interactive dashboard to visualize the findings.
##Dataset Description
The dataset consists of 1,026 records of customer information.
 * Important indicator: Purchased Bike (Yes/No)
 * Key Features: Income, Age, Gender, Education, Occupation, Region, Commute Distance, and Marital Status.

##Key Insights
After analyzing the data, several distinct patterns emerged:
 * The "Middle Age" Market: Customers between the ages of 31 and 45 are the most likely to purchase a bike. This segment represents the highest volume of successful conversions.
 * Commute Influence: There is a strong correlation between short commutes and bike ownership. Customers living within 0–1 miles of their workplace are the primary buyers.
 * Economic Drivers: High-income earners, specifically in Professional and Skilled Manual roles, showed a significantly higher purchase rate compared to manual laborers.
 * Regional Performance: The Pacific region leads in conversion percentage, while North America represents the largest total volume of potential customers.

##Technical Skills Demonstrated
 * Data Cleaning: Handled missing values, standardized categorical labels, and performed data type conversion.
 * Advanced Formulas: Used IFS and nested logic to create dynamic Age Brackets for better segmentation.
 * PivotTables & PivotCharts: Summarized over 1,000 rows of data into actionable visual insights.
 * Dashboard Design: Built a user-friendly interface using Slicers to allow for real-time data filtering by Region, Education, and Occupation.

##How to Use the Dashboard
 * Download the Bike_Buyer_sales_report.xlsx file from this repository.
 * Open the file in Excel.
 * Navigate to the Dashboard sheet.
 * Use the Slicers on the side to filter the charts and observe how demographics affect purchase rates across different regions.

##Project Files
